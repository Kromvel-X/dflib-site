Charts and Dashboards

DFLib helps Java programmers to create a few types of data visualizations. The first is visual charts based on DataFrame data. The charts are generated as JavaScript code using the API from Apache Echarts library and can be viewed in Jupyter notebooks or embedded in an HTML page. Additionally, DFLib can generate "dashboards", i.e. full HTML pages with multiple charts based on built-in or user-provided templates.
